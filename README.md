# energy-detector
五邑大学IMCA战队给辽科的铁子的能量机关分享，此功能包经过测试是能够满足识别要求的（前提是模型训练好）

nanodet-plus进行待打击叶片的识别，提取roi进行传统识别寻找pnp所需的特征点为上下两个等条的内侧角点
